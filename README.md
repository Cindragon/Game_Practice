# Mini-Game-Compilation
This project is a collection of three mini-games(Snake, Doodle Jump, and Tetris) built using the SFML library in C++. Users can select a game to play from the main menu, and their scores will be recorded accordingly.

The primary purpose of this project is to practice and improve C++ programming skills and gain hands-on experience with SFML (Simple and Fast Multimedia Library).
